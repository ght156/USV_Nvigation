# 船体与导航参数索引（实船）

路径相对于 `src/USV_NAV/`。完整键名见 **`workspace_nav/config/boat_parameters_index.yaml`**。

> **Gazebo / URDF / `converter`** 在 **`wuxihik_navigation`**，本仓不维护。

---

## 1. Nav2 船体与避障（实船）

| 文件 | 内容 |
|------|------|
| **`workspace_nav/config/nav2_params_real_mavros.yaml`** | **footprint**、**`VoxelLayer` + `/livox/lidar`**、RPP、**`velocity_smoother`**（`/cmd_vel_nav`→`/cmd_vel`）。 |
| **`launch/nav2_real_mavros.launch.py`** | 默认 **`map_real_boat_hk.yaml`**。 |

---

## 2. 地图文件

| 文件 | 用途 |
|------|------|
| **`config/map_real_boat_hk.yaml`** + **`map/hk_map.pgm`** | **默认实船**（HK 园区） |
| **`config/map.yaml`** + **`map/map.pgm`** | 备用/它区 |

---

## 3. 实船 TF 与传感器杆臂

| 文件 | 内容 |
|------|------|
| **`workspace_ros/config/static_transform_real_boat.yaml`** | `base_link`→`livox_frame`/IMU/GPS 等 |
| **`workspace_ros/config/mavros_px4_overrides_usv.yaml`** | `odom→base_link` |

---

## 4. 速度桥（勿用 converter）

| 文件 | 内容 |
|------|------|
| **`workspace_ros/scripts/nav2_cmd_vel_to_mavros.py`** | `/cmd_vel_nav` → `/mavros/setpoint_raw/local` |

**`converter.py`** 仅 wuxihik 仿真推力；**实船不运行**。
