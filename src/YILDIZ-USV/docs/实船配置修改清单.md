# 实船配置修改清单（按文件）

面向 **真 USV（ROS 2 Nav2 + MAVROS/PX4）**。优先核对 **§1 Nav2**、**§2 定位/TF**、**§3 MAVROS/速度桥**。

**船体 / footprint / converter（仅 wuxihik 仿真）**：[`船体与导航参数索引.md`](./船体与导航参数索引.md)。

**运行模式（本仓默认）**

| 项 | 说明 |
|----|------|
| **定位** | `localization_backend:=mavros_odom`（bringup **默认**） |
| **里程** | `/mavros/local_position/odom` |
| **参考点** | map YAML 的 **`ref_gnss*`** → **`gnss_odom_map_tf`** |
| **EKF+navsat** | **未提供** `localization.launch.py`；勿设 `robot_localization` |

---

## 1. Nav2 与地图（`workspace_nav`）

| 文件 | 要改的内容 |
|------|------------|
| **`config/nav2_params_real_mavros.yaml`** | 实船全参；odom **`/mavros/local_position/odom`**；Livox **`/livox/lidar`**；RPP 与保守速度；**`velocity_smoother`** 参数（Nav2：`/cmd_vel_nav`→`/cmd_vel`）。 |
| **`launch/nav2_real_mavros.launch.py`** | 默认 **`nav2_params_real_mavros.yaml`**、**`map_real_boat_hk.yaml`**。 |
| **`config/map_real_boat_hk.yaml`** + **`map/hk_map.pgm`** | **默认实船海图**；`ref_gnss_*` 为 `[lon, lat]`。 |
| **`config/map.yaml`** + **`map/map.pgm`** | 备用/它区；换图时 bringup **`map_config_yaml:=`** 与 **`map:=`** 须一致。 |
| ~~`nav2.launch.py` / `nav2_params.yaml`~~ | **本仓无**；仿真在 wuxihik。 |

---

## 2. 定位与 TF（`workspace_ros`）

| 文件 | 要改的内容 |
|------|------------|
| **`config/static_transform_real_boat.yaml`** | `base_link`→`livox_frame`/IMU/GPS 等；按标定改。 |
| **`launch/real_boat_bringup.launch.py`** | `use_sim_time:=false`；**`enable_nav2_cmd_vel_to_mavros:=true`**；**`map_config_yaml`** / **`map_origin_ref_key`** 与 Nav2 **`map:=`** 同源。 |
| **`launch/real_boat_mavros_tf.launch.py`** | 单独起 TF 时参数同 bringup。 |
| **`scripts/gnss_odom_map_tf.py`** | 读 **`map_config_yaml`**；算法见 **`实船调试.md`**。 |
| **`config/navsat.yaml` / `ekf.yaml`** | 仅将来补 EKF 链时使用；默认实船 **不读**。 |

### TF 发布者（模式 B）

| TF 段 | 来源 |
|-------|------|
| `map→odom` | `gnss_odom_map_tf` |
| `odom→base_link` | MAVROS + `mavros_px4_overrides_usv.yaml` |
| 传感器 | `static_transform_real_boat.yaml` |

---

## 3. MAVROS 与速度桥

| 文件 | 要改的内容 |
|------|------------|
| **`launch/mavros_px4_usv.launch.py`** | `fcu_url`、插件 YAML 路径。 |
| **`config/mavros_px4_overrides_usv.yaml`** | `local_position` 发 **`odom→base_link`**；`global_position.tf.send: false`。 |
| **`scripts/nav2_cmd_vel_to_mavros.py`** + **`launch/nav2_cmd_vel_mavros.launch.py`** | **`/cmd_vel_nav` → `/mavros/setpoint_raw/local`**；OFFBOARD 门控、限幅。 |
| ~~`converter.py`~~ | **实船不运行**（Gazebo 推力，wuxihik 仿真）。 |

---

## 4. 航点（按需）

| 文件 | 要改的内容 |
|------|------------|
| **`waypoint_transform.py`** | `map_yaml_path`、`map_datum_ref_key` 与 Nav2 海图一致；默认 `datum_source:=map_yaml`。 |

---

## 5. 实船最小启动速记

```bash
ros2 launch workspace_ros mavros_px4_usv.launch.py fcu_url:=/dev/ttyACM0:57600

ros2 launch workspace_ros real_boat_bringup.launch.py \
  use_sim_time:=false enable_nav2_cmd_vel_to_mavros:=true \
  map_config_yaml:=<与 Nav2 相同的 yaml> map_origin_ref_key:=ref_gnss_10

ros2 launch workspace_nav nav2_real_mavros.launch.py map:=<同上>
```
