- **仅「UTM 减 datum」≠ Nav2 `map`**：必须与 **`origin`** 及 **与栅格绑定的 ref 角点**一致，否则目标点落在错误四象限或规划失败。  
- **负坐标期望为正**：多为 **datum 用了错的 `ref_gnss_*`**（与制图 (0,0) 角不一致），改 `map_datum_ref_key` 或修正 `map.yaml` 角点定义。
