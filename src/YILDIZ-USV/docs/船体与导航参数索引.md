# 船体与导航相关参数在仓库中的位置（索引）

本文档回答：**船的尺度、质量、推进/流体、Nav2  footprint、差速映射、仿真/实船地图文件** 等分别在哪改。路径均相对于包根目录（如 `workspace_nav/`、`workspace_gz/`）。完整 YAML 键索引另见 **`workspace_nav/config/boat_parameters_index.yaml`**（含 **`nav2_maps`**）。

---

## 1. 仿真：几何、质量、碰撞、关节（Gazebo / URDF）

| 文件 | 内容 |
|------|------|
| **`workspace_gz/description/roboboat/roboboat.xacro`** | **主体**：`base_link` 的 **质量/惯量**（`<inertial>`）、**collision** 盒子尺寸与位姿、外形 **mesh** 路径。 **传感器杆臂**：`base_link_to_*_link` 的 **`origin xyz/rpy`**（GPS/IMU/雷达/相机/左右推进舱）。 **Gazebo 推进器插件**（`gz-sim-thruster-system`）：**`thrust_coefficient`**、**`fluid_density`**、**`propeller_diameter`** 等。 **水面/阻力**（VRX 插件）：**Surface** 的 **`hull_length`/`hull_radius`/`fluid_level`/`points`**；**SimpleHydrodynamics** 的 **`xU`,`xUU`,`yV`,`yVV`,`nR`,`nRR`** 等阻尼系数。 |

**网格与材质**：`workspace_gz/models/roboboat/`（`meshes/`、`materials/`）。

---

## 2. Nav2：等效船体轮廓与航行”手感”（规划/避障/控制）

| 文件 | 内容 |
|------|------|
| **`workspace_nav/config/nav2_params.yaml`** | 仿真默认。**`local_costmap` / `global_costmap` → `footprint`**：船在水平面投影多边形 (1.0×0.6m，base_link 几何中心)；**`FollowPath`**：`desired_linear_vel`、`lookahead_*`、**`inflation_layer.inflation_radius`** 等。 |
| **`workspace_nav/config/nav2_params_real_mavros.yaml`** | **实船 MAVROS** (1.6×0.8m，保守)。**footprint** 非对称：base_link 在船尾前 0.2m 处，故顶点为 `[[1.4,0.4],[1.4,-0.4],[-0.2,-0.4],[-0.2,0.4]]`。**local**: `voxel_layer` + `inflation_layer`（膨胀 1.2m，mark_threshold=1）。**global**: `static_layer` + `voxel_layer` + `inflation_layer`（膨胀 1.8m，远距离预判 max 40m。注意动态物 ghost 残留风险，岸测可临时关掉）。**控制器**：RPP，`desired_linear_vel=0.2`，前视 1.6~3.5m，弯道减速半径 5.0m，代价缩放距离 2.5m。详细参数注释见文件内 `——` 标记行。配合 **`launch/nav2_real_mavros.launch.py`**（默认 **`map_real_boat_hk.yaml`** + **`hk_map.pgm`**）。 |


**栅格与索引**：仿真 **`map.yaml`** / **`map.pgm`**；实船 HK **`map_real_boat_hk.yaml`** / **`hk_map.pgm`**；备份 **`map.simulation.backup.*`**；完整键见 **`workspace_nav/config/boat_parameters_index.yaml`** 的 **`nav2_maps`**。

## 3. 执行层：差速推力映射（ cmd_vel → 左右推力，仿真桥）

| 文件 | 内容 |
|------|------|
| **`workspace_ros/scripts/converter.py`** | **`linear_scale`**、**`angular_scale`**（`Twist` 到左右 **`/roboboat/thrusters/.../thrust`** 的线性组合）；**推力话题名**亦可改。 |
| **`workspace_ros/scripts/manual_control.py`** | 键盘试推时 **左右推力常量**（如 `±10.0`）。 |

真船若走 MAVROS/PX4，推力一般不在此两项，而在 **飞控混控参数**（仓库外 QGC）。

## 4. TF：传感器相对船体安装

| 文件 | 内容 |
|------|------|
| **`workspace_ros/config/static_transform.yaml`** | **仿真及 `localization.launch.py`（模式 A）**：`*_link` → **`roboboat/base_link/sensor_*`**，与 Gazebo 侧命名配套。 |
| **`workspace_ros/config/static_transform_real_boat.yaml`** | **实船模式 B**（**`real_boat_mavros_tf`** 默认；旧入口 **`real_boat_tf_static`**）：**`base_link`** → `lidar_link` / `imu_link` 等，外参默认同 **`roboboat.xacro`**，真船按测量改。 |
| **`real_boat_bringup` → `gnss_odom_map_tf`** | **`map_config_yaml`** 必须与 **`nav2_real_mavros.launch.py`** 的 **`map:=`** **同一文件**；**`map_origin_ref_key`** 指明 **`ref_gnss*`** 中哪一个角对应 Nav2 **`map`** 原点 **(0,0)**（默认 **`ref_gnss_10`**）。详见 **`docs/实船调试.md`**。 |

**复盘（谁发哪段 TF、`view_frames` 等）**： **[`实船调试.md`](./实船调试.md)**「TF 坐标系总览与复盘调试」、[`PROJECT_ARCHITECTURE_AND_NAV2.md`](./PROJECT_ARCHITECTURE_AND_NAV2.md) §4.0。

## 6. 真船 MAVROS（动力学不在本仓库）

**PX4：螺旋桨混控、PWM、艇体控制律** 等在 **地面站 / 飞控固件参数**，不在本 ROS 仓库；本仓库桥接话题（如 **`nav2_cmd_vel_to_mavros`**）、Nav2，以及 **MAVROS ROS 2 参数覆盖**。

| 文件 | 内容 |
|------|------|
| **`workspace_ros/launch/mavros_px4_usv.launch.py`** | 实船推荐入口：按顺序加载 **`px4_pluginlists.yaml`** → **`px4_config.yaml`**（随 `mavros` 安装）→ **`config/mavros_px4_overrides_usv.yaml`**，再传入 **`fcu_url`** / **`gcs_url`** 等。 |
| **`workspace_ros/config/mavros_px4_overrides_usv.yaml`** | **`local_position`**：**`tf.send: true`**、**`frame_id`** / **`tf.frame_id`** 与 **`tf.child_frame_id`** 对齐 **`odom`** / **`base_link`**；**`global_position.tf.send: false`** 等减轻与 **`map`** 链冲突。按需在此追加键，勿整份拷贝 upstream。 |

---

## 7. 修改时的建议顺序（避免漏项）

1. **真船尺寸 / 激光避障**：改 Nav2 **`footprint`** + **激光外参**（实船改 **`static_transform_real_boat.yaml`**，仿真改 **xacro joint** 与 **`static_transform.yaml`**）。  
2. **仿真与真船一致的外形**：同步改 **`roboboat.xacro` 碰撞体** 与 **Nav2 footprint**（量级一致）。  
3. **推力/加速度手感**：仿真调 **converter 比例 + xacro 推进器/流体插件**；实船调 **PX4** + Nav2 **速度/前瞻**。
